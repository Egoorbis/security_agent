# Security Agent

A comprehensive, **multi-language security analysis agent** that detects programming languages, adapts security scanning strategies, and assesses compliance against industry best practices including OWASP, Microsoft Azure, Terraform, Python, .NET, Java, and MCP security standards.

## Quick Start

Choose your setup method:

- **🤖 [VS Code Agent Integration](VS_CODE_AGENT_SETUP.md)** - Configure as GitHub Copilot agent or VS Code extension
- **🛠️ [Local VS Code Setup](LOCAL_VS_CODE_SETUP.md)** - Run the security agent locally on any repository you have open in VS Code
- **🚀 [Reusable GitHub Actions Workflow](REUSABLE_WORKFLOW_USAGE.md)** - Integrate security scanning into your CI/CD pipeline across multiple repositories
- **💻 [GitHub Actions Configuration](.github/README.md)** - Technical details about the workflow structure
